# Quickstart: APIリクエストUI

このUIは image-saver の内部APIを同一オリジンで手早く確認するための補助ツールです。

## 前提
- APIとUIは同一オリジンで提供されます
- 本UIは本プロジェクトのAPIのみ対象です（外部任意APIは対象外）
- 接続設定（ベースURL/共通ヘッダー/タイムアウト）は端末ローカルに保存されます（明示リセット可）

## 使い方（想定）
1. ベースURLを入力（初回のみ）
2. メソッド/パス/クエリ/ヘッダー/本文(JSON)を指定
3. 送信して結果を確認（ステータス・時間・本文）
4. よく使う操作はプリセットから選択して短縮

## エラーと対処
- 入力エラー: JSON整形/バリデートを行い、不正箇所を表示
- 接続エラー: タイムアウト/ネットワーク断等を分類して解決のヒントを表示
- サーバーエラー: ステータスと本文を見やすく表示（ログは既存API側）

## セキュリティ注意
- トークン等の機微情報はマスク表示されます
- 端末ローカル保存が有効のため、共有端末では利用後にリセットしてください
